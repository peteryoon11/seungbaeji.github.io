머신러닝 블로그 시작!

2018-09-16
#일상
일반

여러가지 블로그 서비스를 살펴보다가, hexo + github으로 블로그를 운영하기로 결정하였다. 선택하게된 가장 큰 이유는 mathjax를 이용할 수 있기 때문이다.

머신러닝쪽 정리를 하게되다 보면, 수식입력은 필수적인 요소이다. 하지만 대부분의 블로그 서비스는 수식입력을 지원하지 않으며, 지원할 계획이 없다고 한다. (medium을 쓰고 싶었는데…)

결국 선택지는 jekyll과 hexo 이렇게 두 가지가 남았으며, 사용의 편의성을 고려하여 마크다운이 활용 가능한 hexo로 최종 결정을 하였다. 그리고 블로그 설정에는 링크1, 링크2, 링크3을 참조하여 설정을 하였다.

마지막으로 해당 블로그는 서버 구축, Scikit-learn, Tensorflow 등의 Python 생태계에서의 머신러닝을 전반적으로 다룰 것이다. 그리고 개인적으로 흥미가 많은 sequential data와 coin data를 집중적으로 탐구해 보려 한다.

p.s. Latex 문법을 이용하여 아래와 같이 수식 표현이 가능하다.
$$P(a|b) = \frac{P(b|a)P(a)}{P(b)}$$
